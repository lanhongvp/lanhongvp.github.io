pip换源

小白如我，在不知道 pip换源 这一操作之前，安装一个包，看一集剧（偷懒的绝佳借口）
但是，自从换了源，就再也不能在 pip安装 的时候愉快滴看剧了，妈妈再也不用担心我的学习。
之所以要换源，就是因为原本pip安装的默认网址是国外的网站，所以速度灰常慢，还经常不稳定导致安装失败，但是如果换成国内镜像，速度就可以快到飞起。闲话不多说，看看怎么弄的吧。

  • 进入根目录

    cd ～/

  • 进入.pip目录

    cd .pip

  • 若不存在就新建一个呗，然后再进入.pip目录

    mkdir .pip

    cd .pip

  • 创建pip的配置文件pip.conf(configure)

    touch pip.conf

  • 运用vim修改pip.conf

    vim pip.conf

最后，如果想要通过 pip安装什么包，可以直接运行以下命令（以安装tensorflow1.8为例）

  • python2

    pip install tensorflow==1.8

  • python3

    pip3 install tensorflow==1.8
